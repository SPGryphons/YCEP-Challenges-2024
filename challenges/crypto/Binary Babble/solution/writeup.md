# Solution

1. **Notice there is a pattern in the message.txt**

2. **Convert all `ba` to -**

3. **Convert all `na` to .**

4. **Convert all `no` to " "**

5. **Decode the morse code.**

```
-.-- -.-. . .--. ..--- ....-  .-- ...-- ..--.- .-.. ----- ...- ...-- ..--.- -- ----- .-. ... . ..--.- -.-. ----- -.. ...-- ..--.- -- .-. ...-- -. .-- .. -....
```

We get `W3_L0V3_M0RSE_C0D3_MR3NWI6`

Wrap it to the flag format we got:

Flag: `YCEP24{W3_L0V3_M0RSE_C0D3_MR3NWI6}`
